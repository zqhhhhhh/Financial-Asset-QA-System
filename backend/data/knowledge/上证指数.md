# 上证综合指数

上海证券交易所综合股价指数（简称上证指数、上证綜指、上證綜合、沪綜指或沪指，英語：SSE Composite Index），上海證券交易所主要的綜合股價指數，为一种市值加权指数，是反應挂牌股票总体走势的统计指标。

历史
上证指数的前身为上海静安指数，最初是中国工商银行上海市分行信托投资公司静安证券业务部于1989年初根据上海股市的实际情况、参考国外股价指标的生成方法编制而成。上海证券交易所成立后，静安证券业务部当时所从属的申银证券公司将资料移交上海证券交易所，上海证券交易所以当时移交的八只股票为样本、以1989年12月19日为基期编制了上证指数，并于1991年7月15日起正式发布。
1992年2月21日，增设上证A股指数与上证B股指数。1993年6月1日，又增设了上证分类指数，即工业类指数、商业类指数、地产业类指数、公用事业类指数、综合业类指数、以反映不同行业股票的各自走势。
2020年6月19日，上交所宣佈將從7月22日起修订上证综合指数编制方案，ST、*ST股從指數中剔除，延长新股计入指數时间，指數纳入科创板上市证券。

指數歷史
最高與最低是以交易中出現的極值為準。
