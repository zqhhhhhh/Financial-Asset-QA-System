# 交割日

交割日（英語：Settlement date）是證券業中的一個術語，用來描述一筆交易（如債券、股票、外匯、商品等）完成交割的日期。也就是說，這是現金或資產實際完成轉移的日期，通常會發生在交易完成之後的數天。
交易日與交割日之間的天數，取決於該證券種類及其所屬市場的慣例。例如，在倫敦證券交易所進行股票交易時，交割通常設定為交易日後第2個工作日（T+2）。在美國，交割期間已於2017年由 T+3 縮短為 T+2，並於2024年進一步縮短為 T+1。
交割日不一定等同於起息日／價值日（Value Date）（即用來計算結算金額的日期）。例如，後台作業可能需要數天時間來完成付款處理。這種估值與實際交割之間的時間差，通常會寫入金融契約中；此外，實際交割日也可能因為問題或錯誤，而與原先約定的日期不同。
此日期有時也被稱為「契約交割日」（Contractual Settlement Date）。
