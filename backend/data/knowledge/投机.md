# 投机

投機（英語：speculation），不同於投資，投機存在較高的風險，投機希望的是能在短期獲得的利益，以及炒作的方式介入其中謀取利益。常見的手段炒賣是用貨幣所有者以其所持有的貨幣購入非貨幣資產，然後在未來將購得的非貨幣資產再次轉換為貨幣資產，以賺取較低的購入價格和較高的出售價格之間的利潤，即差價。或者用財務槓桿的方式來放大利潤。
投機者同以抵消頭寸風險為目的的對沖者、以獲取不同市場中同類產品差價為目的套利者以及以通過長期持有而獲取利潤為目的的投資者共同構成金融市場的四大主要參與者。

炒賣
炒賣是投機者意圖在短時期內從低價購入再從高價賣出轉售圖利，常見被炒賣的有股票、樓房、黃牛票等的差價，但投資與投機行為不是互斥的，也可以同時存在，這端看資產的實際價值與持有狀態而定，而非在低買高賣的行為定義。

資金槓桿
投資者尋求的是無風險或接近零風險的情況下換取穩定的回報，投機者追求的是拉高風險換取較多的回報。投資者常會選擇不槓桿或者低槓桿的方式來投資，以降低風險。而資金槓桿對於投機者來說是經常選擇的手段之一，透過拉高風險的方式來取得更多的利益，甚至會用盡可能將槓桿開到最大以換取最大的利益。房地產是常見的資金槓桿市場，由於是抵押品來承作擔保的關係，銀行更願意給出大筆金額，所以允許透過抵押品的方式來換取較大的槓桿率，但這也增加的泡沫經濟的風險。
