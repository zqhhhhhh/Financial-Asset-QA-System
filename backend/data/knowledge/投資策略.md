# 投資策略

在金融领域，投资策略是一種指南，旨在指导投资者选择適合的投資組合。每个人根據各自的獲利目标選擇合適的投資組合。有些投資者選擇投資組合時會考慮如何平衡风险與收益。大多数投资者為获得更高的回报會接受一定的投資风险。

概述
在经济学领域，这一决策由寻找具有最大效用的投资策略所驱动。随着投资策略日益复杂化，公司开始使用基于人工智能的资产管理工具来提高决策效率并优化流程。投资者经常选择投资来对冲通胀。在高通胀时期，诸如股票之类的投资在通胀调整后通常表现更差。
策略类型:

无策略。普林斯顿大学的一位教授提出了这样一个理论：“一个蒙着眼睛、向金融版报纸投镖的猴子，可能会挑选出一个与专家精心挑选的投资组合具有相同表现的组合”。这一结论存在争议。
主动与被动策略: 被动策略，如“买入并持有”以及被动指数化，常用于尽量减少交易成本。被动投资者不相信能够准确预测市场走向。主动策略，例如基于动量的交易，试图超越基准指数。主动投资者认为自己的技能高于平均水平。
基于动量的交易: 一种策略是根据资产近期的历史表现来选择投资。过去 3-12 个月收益较高的股票，通常在随后的几个月内相比过去 3-12 个月收益较低的股票仍会继续表现更好。针对这一策略既有支持的证据，也有反对的证据。
