# EBIT(息税前利润)

息稅前利潤（英語：Earnings Before Interest and Taxes，縮寫：EBIT），或譯稅前息前利潤、除稅及利息前盈利，是一家公司在扣除利息、所得稅之前的利潤。息稅前利潤與營業利潤差異在於營業利潤不包括營業外收入、營業外支出或其他收入。

算法
净营收（或净销售) = 营业收入（或毛销售) - (顧客折扣 + 退回 + 折讓)
毛利润 = 净营收（或净销售) - 營業成本（或销售成本）
营业利润 = 毛利润 - 營業費用  
息稅前利潤 = 营业利润 + 营业外利润（即營業外收入 - 營業外支出）
纯利 = 息稅前利潤 - 利息支出 - 稅 = 税前利润 - 稅 
即：
纯利 = 净营收 - 營業成本  - 營業費用 + 营业外利润 - 利息支出 - 稅
