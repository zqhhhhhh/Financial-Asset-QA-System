# 清算 (金融)

清算是一个在银行和金融领域的術語，是指从对交易作出承诺到结算为止的所有活动。此过程将付款承诺（例如，以支票或电子付款请求的形式）转化为资金从一个帐户到另一个帐户的实际流动。
