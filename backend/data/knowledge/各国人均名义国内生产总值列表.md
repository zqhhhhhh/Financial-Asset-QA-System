# 各国人均名义国内生产总值列表

本條目為各國家或地區的人均名义国内生产总值（GDP）列表，列出各國家或地區的人均名义GDP（台湾稱作人均名目GDP，英語：GDP nominal per capita），並且按官方美元匯率所估算。
此處提供的數字未考慮不同國家/地區生活成本的差異，並且根據國家/地區貨幣匯率的波動，結果每年都有很大差異。 這種波動會改變一個國家的排名，從一年到下一年，儘管它們通常對其人口的生活水平影響很小或沒有影響。因此，應謹慎使用這些數字。 人均國內生產總值通常被認為是一個國家生活水平的指標； 然而，這是不準確的，因為人均GDP不是個人收入的衡量標準。
國内生產總值是一个經濟體内所有居民生產者創造的增加值的總和，加上產品税，並減去不包括在產品價值中的補貼。計算時未扣除資產折扣或自然資源損耗和退化。人均GDP是國内生產總值除以年中人口數。
這裡的數字沒有考慮到生活費用在不同的國家的差異，而且因為匯率變化每年其結果可以差别很大。对于名义GDP数据，這種匯率波動可能會改變一個國家的每年排名，即使他們往往很少或根本沒有改變其人民的生活水準。因此，這些數字應謹慎使用，否則有可能會在應用時發生錯誤。
比較國家財富也經常以購買力平價（PPP）作為基础，來調整各國不同的生活費用（詳見各國人均實質GDP列表）。購買力平價，主要是删除了滙率的問題，但有它自己的缺點，即并不反映經濟總量在國際貿易中的價值，比起來名义人均國内生產總值，它也需要更多的估計數字，因此沒有名义人均國內生產總值使用得廣泛。
請注意，幾個領先的人均GDP（名義）司法管轄區可能被視為避稅天堂，它們的GDP數據可能會因稅收籌劃活動而受到嚴重扭曲。 例子包括百慕大、開曼群島和盧森堡。

列表
该表默认以國際貨幣基金組織（IMF）公布的各国家或地区的最新估计值进行排名。如果IMF未能提供某个国家或地区的最新数据，则使用世界银行或联合国提供的最新数据排名。您也可选择列表中的图标以选择其中一个来源重新排名。
最左栏的排名数字不包括地区（在列表中使用 背景色 标记）。
