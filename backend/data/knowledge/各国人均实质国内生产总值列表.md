# 各国人均实质国内生产总值列表

本條目為各國家或地區的人均國內生產總值（GDP）經購買力平價（PPP）調整後排序的列表。

列表
所有数字均以目前的国际元计算，并四舍五入到最接近的整数。
该表預設以國際貨幣基金組織（IMF）公布的各国家或地区的最新估计值进行排名。如果IMF未能提供某个国家或地区的最新数据，则使用世界银行或联合国提供的最新数据排名。您也可选择列表中的图标以选择其中一个来源重新排名。
* 表示链接到该国经济的条目
